Perguntas e Exemplos MySQL

  1. Como criar um banco de dados no MySQL?
    CREATE DATABASE nome_do_banco;
  2. Qual comando é usado para selecionar um banco de dados específico?
    USE nome_do_banco;
  3. Como criar uma tabela no MySQL?
    CREATE TABLE nome_da_tabela ( coluna1 tipo, coluna2 tipo, ... );
  4. Qual é o comando para inserir dados em uma tabela?
    INSERT INTO nome_da_tabela (coluna1, coluna2, ...) VALUES (valor1, valor2, ...);
  5. Como recuperar todos os registros de uma tabela?
    SELECT * FROM nome_da_tabela;
  6. Qual é o comando para filtrar dados em uma consulta?
    SELECT coluna1, coluna2 FROM nome_da_tabela WHERE condição;
  7. Como atualizar dados em uma tabela?
    UPDATE nome_da_tabela SET coluna1 = novo_valor WHERE condição;
  8. Como excluir registros de uma tabela?
    DELETE FROM nome_da_tabela WHERE condição;
  9. Como adicionar uma chave primária a uma tabela já existente?
    ALTER TABLE nome_da_tabela ADD PRIMARY KEY (coluna);
  10. Como realizar uma junção (JOIN) entre duas tabelas?
    SELECT * FROM tabela1 JOIN tabela2 ON tabela1.coluna = tabela2.coluna;
  11. Qual é a função do GROUP BY?
    SELECT coluna, COUNT(*) FROM nome_da_tabela GROUP BY coluna;
  12. Como ordenar os resultados de uma consulta?
    SELECT * FROM nome_da_tabela ORDER BY coluna DESC;
  13. Como calcular a média de uma coluna numérica?
    SELECT AVG(coluna_numerica) FROM nome_da_tabela;
  14. Como usar a cláusula LIMIT para paginar resultados?
    SELECT * FROM nome_da_tabela LIMIT 10 OFFSET 20;
  15. Como realizar uma subconsulta (subquery)?
    SELECT coluna FROM nome_da_tabela WHERE coluna IN (SELECT coluna FROM outra_tabela);
  16. Como fazer backup e restaurar um banco de dados?
    - Backup:
    mysqldump -u usuario -p nome_do_banco > backup.sql
    - Restaurar:
    mysql -u usuario -p nome_do_banco < backup.sql
  17. Como criar um índice em uma coluna para melhorar o desempenho de consultas?
    CREATE INDEX index_nome ON nome_da_tabela (coluna);
  18. Como usar transações no MySQL?
    START TRANSACTION; -- Suas operações SQL aqui COMMIT;
  19. Como renomear uma tabela?
    RENAME TABLE nome_antigo TO nome_novo;
  20. Como contar o número de registros em uma tabela?
    SELECT COUNT(*) FROM nome_da_tabela;
